Data Dictionary: Stack Exchange

The raw Stack Exchange data is organized by their top-level domains in the Stack Exchange network. For each domain, there are eight types of data organized into separate XML files.

  • Badges
  • Comments
  • PostHistory
  • PostLinks
  • Posts
  • Tags
  • Users
  • Votes

Each top-level domain is represented as a folder in the SayWhat S3 bucket. Each of those folders contains an XML file with information for each of the above eight topics.
